# Features List _ Res Hub

## 1. Core Features 

### 1.1 User Registration & Authentication 
• Verified student registration using institutional email (e.g., WSU email) 
• Secure login and logout functionality 
• Password reset and account recovery 
• User profile management (name, student number, faculty, year) 

### 1.2 Residence Review System 
Students can submit structured reviews for residences they have lived in 
Reviews include: 
• Overall rating (e.g., 1–5 stars) 
• Written feedback (experience, pros, cons) 
• Date of stay (to ensure relevance) 
• Ability to edit or delete own reviews 
• Prevention of duplicate reviews per residence per student 

### 1.3 Landlord Rating System 
Students can rate landlords separately from the residence 
Rating criteria may include: 
• Responsiveness 
• Professionalism 
• Fairness 
• Maintenance handling 
• Optional written feedback on landlord behavior 
• Aggregated landlord ratings visible to all users

### 1.4 Residence Rating Breakdown 
Multi-criteria rating system: 
• Infrastructure quality 
• Maintenance 
• Safety (location & security) 
• Cleanliness 
• Accessibility (transport, campus distance) 
• Display of average ratings per category 
• Overall score automatically calculated 
 
### 1.5 Residence Search & Discovery 
Search residences by: 
• Name 
• Location 
• Price range 
Filter by: 
• Rating 
• Safety score 
• Distance from campus 
Sort by: 
• Highest rated 
• Most reviewed 
• Recently added 

### 1.6 Review Transparency & Verification 
• Only verified students can post reviews 
• Badge/indicator for verified reviewers 
• Display number of reviewers per residence 
• Flag/report inappropriate or fake reviews 

## 2. Supporting Features 

### 2.1 Aggregated Insights Dashboard 
Summary statistics per residence: 
• Average rating 
• Common complaints 
• Most mentioned positives 
• Visual charts (optional enhancement) 

### 2.2 Favorites / Bookmarking 
• Users can save residences for later viewing 
• Quick access to shortlisted accommodations 

### 2.3 Notifications System 
Notify users when: 
• Their review receives interaction 
• Maintenance issue status changes 
• New reviews are posted on saved residences 

### 2.4 Admin Panel 
• Manage users (approve, suspend, delete) 
• Moderate reviews and reports 
• Manage residence listings 
• Monitor flagged content 

## 3. Non-Functional Features 

### 3.1 Security 
• Secure authentication (hashed passwords) 
• Role-based access (student vs admin) 
• Protection against fake accounts 

### 3.2 Usability 
• Simple and intuitive user interface 
• Mobile-friendly design 
• Easy navigation and accessibility 

### 3.3 Performance 
• Fast loading of reviews and ratings 
• Efficient search and filtering 
• Scalable database design 

### 3.4 Reliability 
• High system availability 
• Data consistency and integrity 
• Backup and recovery mechanisms
