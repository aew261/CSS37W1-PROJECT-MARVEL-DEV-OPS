# User Stories

- As a user (student), i want to be able to search for accommodation so i can quickly view accomodation i desire.
- As a user (student), i want to be able to be recommended residences then ones with high ratings so i can choose residence's with the best quality and best services.
- As a user (student), i want to be able to write a review about a residence if i stayed i one before so others considering the residence can know my experience.
- As a user (student), i want to be able to to rate services provided by the accomodation if i lived in it previously.
- As a user (student), i want to be able to rate or like reviews by other students so they can know how heplful their insight has been.
- As a user (student), i want to be able to view preview images of the residence check out the infrastructure.
- As a user (student), I want to be able to view past reported issues in my residence so I can understand recurring problems.
- As a user (student), I want to receive notifications when my review gets liked or replied to.
- As an admin, I want to hide or delete inappropriate reviews, so the platform stays clean.
- As an admin , i want to upload residence listings and descriptions to avoid fake listings.
- As an admin, I want to be able to view all user-reported issues so I can monitor platform activity.
- As an admin, I want to be able to review flagged or reported content so I can take appropriate action.
- As an admin, I want to be able to issue warnings or sanctions to users who violate guidelines so I can maintain discipline.
