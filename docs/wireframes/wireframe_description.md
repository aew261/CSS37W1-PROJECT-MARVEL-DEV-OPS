##RES HUB Login Screen
 
Header: House icon logo with "RES HUB" text at the top
Title: "Login or sign up" heading
Username field: Text input box with a person icon on the right
Password field: Text input box with a lock icon on the right
Login button: Blue button to submit credentials
Forgot password link: Gray text option for password recovery
Sign-up button: Blue button to create a new account
 
Color scheme: Dark blue background with white/light gray input fields and blue action buttons
The layout is centered and straightforward, designed for users to either log in with existing credentials or sign up for a new account.
